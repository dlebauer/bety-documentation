Tables

The database is designed as a relationship database management system (RDBMS), following the normalization fig:summarymodel. Each record in each table has a unique identifier for each record. This is the the primary key field, id, which is described below. In addition, each record has created_at and updated_at fields. The traits and yields tables each has a user_id field to record the user who originally entered the data.
