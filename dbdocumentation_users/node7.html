How to download data

Data can be downloaded as a .csv= file, and data from previously published syntheses can be downloaded without login. For example, to download all of the Switchgrass (Panicum virgatum L.) yield data,

  1. Open the BETY homepage ebi-forecast.igb.uiuc.edu/betyebi-forecast.igb.uiuc.edu
  2. Select http://ebi-forecast.igb.uiuc.edu/bety/maps/species_detailsSpecies database under Search
  3. Select http://ebi-forecast.igb.uiuc.edu/bety/maps/yields?species=938Click Here under Yields
  4. to download all records as a comma-delimited (.csv) file, scroll down and select the link http://ebi-forecast.igb.uiuc.edu/bety/maps/yields?format=csv&species=938CSV Format
