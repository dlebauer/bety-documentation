Data Entry

The http://dl.dropbox.com/u/18092793/dbdocumentation_data_entry.pdfData Entry Workflow provides a complete description of the data entry process. BETY's web interface has been developed to facilitate accurate and efficient data entry. This interface provides logical workflow to guide the user through comprehensively documenting data along with species, site information, and experimental methods. This workflow is outlined in the BETYdb Data Entry. Data entry requires a login with Create permissions, this can be obtained by contacting mailto:dlebauer@illinois.eduDavid LeBauer or mailto:mdietze@illinois.eduMike Dietze.
