Citations_Sites

Because a single study can use multiple sites and multiple studies can use the same site, these relationships are tracked in the Citation_has_Site table (Table ).

To record one or more covariates associated with each data point, there is a covariates (Table 6). For example, photosynthesis measurements are often recorded along with irradiance, temperature, and relative humidity. Covariates are frequently recorded for multiple data points, so an m:n lookup table was required. The definition of specific covariates can be found in the variables 20 table.
